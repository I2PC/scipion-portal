A fast way to reach cryo-em users.

Scipion is a plugin framework ready to discover new EM image processing software once you wrap your software as a scipion plugin.

Developer docs

Written in python

Scipion is written python and its plugins too. Python has nowadays gained popularity being one of the most popular programming language.

Using pypi

We use pypi.org to deploy our plugins and you will be in control of the integration at any time. If the documentation is enough, you might need to contact us to develop a full functional plugin.

At your pace

We are all busy, we know that. That is why we think you should own the whole development cycle of the plugin, at your pace.

Reach SB users fast

Scipion is downloaded 2-3 a day. You can reach all Scipion users as soon as you publish your plugin. See the download statistics.

Automated testing

You code, even your binaries, can be regularly tested. Devel and production branches!

Usage stats

We are collecting usage statistics, with user consent, and you will have feedback about the usage of your methods "out of the box".

Just your method/s

There is no need to wrap your software with 3rd party software to create another "full-stack software suite" to cover the whole pipeline. We are doing it for you!

File format convertions

Common file formats conversions are provided (mrc, vol, hdf, dm4, mrc4bit, tiff, â€¦). You can implement just one and use Scipion/Xmipp file format conversion methods

Focus on the method

Scipion provides a uniform GUI. You just need to focus on the method and, if apply, presenting your nice results as charts, 3D volume rendering or wherever your creativity takes you.
