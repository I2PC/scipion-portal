Contact Scipion core

Mailing list
The easiest way to reach us is by writing to our mailing list: scipion-users@lists.sourceforge.net. You can also subscribe to the list here, to get all the notifications.
Something more private?
If your reason to contact us is private, then reach us at scipion@cnb.csic.es.
Support for facilities
We have a slack workspace for facilities where more sensible or critical conversations can happen. We can help you to install Scipion there and you will be able to chat with the developers. Request access at scipion@cnb.csic.es.
Support for developers
As for facilities we can invite you to our slack workspace. That way you will get almost immediate answers to your "dev-doubts" Request access at scipion@cnb.csic.es.
Creating issues in GitHub
If you have GitHub account (It is very quick to create a new one), you can Open a New Issue to report a bug or suggest a new feature.
